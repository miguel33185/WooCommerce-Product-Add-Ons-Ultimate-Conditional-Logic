== Description ==

WooCommerce Product Add Ons Ultimate Conditional Logic allows your users to customise products through additional fields.
Add extra fields to WooCommerce products and add various filters by product attributes.

== Installation ==
1. Upload the `woocommerce-product-add-ons-ultimate-conditional-logic` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start adding Product Add-Ons in your WooCommerce products
